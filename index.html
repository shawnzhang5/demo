The most interesting project I have worked on is a collaborative project for a mobile and laptop application based on the JDK platform. The goal of this project is to implement an intelligent spreading optimization technique using evolutionary algorithms. These algorithms are designed to promote uniform spreading of nodes, which can represent anything from robots to automobiles across any enclosed landscape. This project has far-reaching impacts across a spectrum of technologies from military defense to epidemiology. Within the spreading algorithms, intelligent nodes assess their surrounding environment, and calculate the most efficient path of travel from those given conditions. The complexity lies in selecting the most efficient route for spreading. The overall difficulty in discovering the best route is the great number of potential solutions in developing the most efficient spreading algorithm. Therefore, to reduce the computation time, a tool called fitness function is applied to path evaluations, making the algorithms more efficient. The management experience I obtained from this project is fascinating as well. The design team has to run a massive field test with more than twenty people. Each individual representing a node has access to the application which is going to be connected to the central access point. A central terminal which contains server side software processes communication between nodes and calculates the fitness function. As a team member I have to instruct participants operation procedures of the software. I coordinate and guide their movements to the appropriate positions marked on the floor. This simulation provides me an overview of spreading pattern and node distribution in scale. Experience from this project does not only benefit from people management but also from asset management. We evaluate and discuss the feasibility and cost for each stage, then make optimized decisions. For instance, we upgrade the most cost-efficient router that enables full coverage of all possible nodes.
